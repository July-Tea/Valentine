# Contributing Guide

感谢你愿意贡献这个项目 💗

## 提交前检查

1. 确保代码可读、变更聚焦
2. 本地运行通过：

```bash
npm run build
```

3. 如果改动交互行为，请补充说明（文字 / GIF / 截图）

## 提交流程

1. Fork 本仓库并创建分支
2. 完成改动后提交 commit
3. 发起 Pull Request，并在描述中说明：
   - 改了什么
   - 为什么改
   - 如何验证

## Issue 建议

提交 Issue 时请包含：

- 复现步骤
- 实际表现
- 期望表现
- 设备和浏览器信息

## 代码风格

- 使用 TypeScript
- 保持配置项与注释清晰
- 避免无关的大范围重构
